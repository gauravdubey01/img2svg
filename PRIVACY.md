# Privacy Policy for Image to SVG Converter

**Last updated:** July 15, 2026

## Data Collection

Image to SVG Converter **does not collect, store, transmit, or share any personal data or usage information.**

The application:

- **Does not** require an internet connection to function (the only optional external link is a donation page that opens in your browser if you choose to click it).
- **Does not** collect analytics, telemetry, or crash reports.
- **Does not** track your usage or behavior.
- **Does not** upload your images or processed files to any server.
- **Does not** store any personal information.

## Local Storage

The only data stored locally on your device is:

- A small flag file (`~/.img2svg_tutorial_done`) that records whether you have completed the first-launch tutorial. This file is stored only on your machine and can be deleted at any time without affecting app functionality.
- SVG files you explicitly export via the "Save SVG" button — these are saved to the location you choose.

## Third-Party Services

The app contains an optional "Support" button that opens the URL [https://ko-fi.com/gauravdubeypro](https://ko-fi.com/gauravdubeypro) in your default web browser. This action is only triggered by your explicit click and is governed by Ko-fi's own privacy policy.

If this policy changes in the future, the updated version will be posted at this URL.
